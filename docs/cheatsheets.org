* Cheatsheets
** Windows
*** Directories to log/monitor
| Directory Path                                                            | Reason                                                              |
|---------------------------------------------------------------------------+---------------------------------------------------------------------|
| C:\Windows\System32\config\SAM                                            | Stores local password hashes                                        |
| C:\Windows\repair\SAM                                                     | Backup of user credentials                                          |
| C:\Windows\System32\config\SECURITY                                       | Holds security policies and access control settings                 |
| C:\Windows\System32\winevt                                                | Stores Windows Event Logs                                           |
| C:\Windows\System32\config\SYSTEM                                         | Tracks system-wide changes and configurations                       |
| C:\Windows\System32\SOFTWARE                                              | Registry hive with details on installed software and changes        |
| C:\Windows\Prefetch                                                       | Tracks recently executed programs                                   |
| C:\Windows\AppCompact\Programs\Amcache.hve                                | Logs details of executed applications                               |
| C:\Users\*\NTUSER.dat                                                     | Holds user-specific registry settings, often abused for persistence |
| C:\Users\*\AppData\Roaming\Micrososft\Windows\Start\Menu\Programs\Startup | User-specific persistence mechanisms                                |
| C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup              | Global startup for all users                                        |
*** Useful Repair Commands
| Command                                      | Reason                                            |
|----------------------------------------------+---------------------------------------------------|
| sfc /scannow                                 | Scans and repairs filesystem                      |
| DISM /Online /Cleanup-Image /RestoreHealth   | Repairs Windows image and recovery                |
| chkdsk C: /f /r                              | Checks for bad sectors of drive letter            |
| net stop wuauserv                            | Stops a Windows Update                            |
| net stop bits                                | Stops the background intellegent transfer service |
| del %winddir\\SoftwareDistribution\*.* /s /q | Deletes corrupted Windows Update cache files      |
| net start weauserv                           | Deletes corrupted Windows Update cache files      |
| net start bits                               | Restarts the BITS service                         |

** RegEx
*Character classes*
| =.=        | Any character except newline  |
| =\w \d \s= | Word, Digit, Whitespace       |
| =\W \D \S= | *NOT* Word, Digit, Whitespace |
| =[abc]=    | Any of a, b, or c             |
| =[^abc]=   | *NOT* a, b, or c              |
| =[a-g]=    | Character between a & g       |
| =[b=q]=    | Character between b & q       |
*Anchors*
| =^abc$= | Start / end of string |
| =\b=    | Word boundary         |
| =\B=    | *NOT* Word boundary   |
*Escaped characters*
| =\. \* \\= | Escaped special characters     |
| =\t \n \r= | Tab, Linefeed, carriage return |
*Groups & Lookaround*
| =(abc)=   | Capture group             |
| =\1=      | backreference to group #1 |
| =(?:abc)= | Non-capturing group       |
| =(?=abc)= | Positive lookahead        |
| =(?!abc)= | Negative lookahead        |
*Quantifiers & Alternation*
| =a*=     | 0 or more              |
| =a+=     | 1 or more              |
| =a?=     | 0 or 1                 |
| =a{5}=   | Exactly five           |
| =a{2,}=  | Two or more            |
| =a{1,3}= | Between one and three  |
| =a{5,7}= | Between five and seven |
-  =ab|cd=     match ab or cd
